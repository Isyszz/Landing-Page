# 🌐 Landing Page — Site de Portfólio Pessoal

Projeto de Landing Page desenvolvido como portfólio pessoal com botão de contato via WhatsApp. Desenvolvido com HTML, CSS e JavaScript como parte do aprendizado em Desenvolvimento Web.

---

## 🖥️ Preview

🔗 [Acesse o projeto aqui](https://isyszz.github.io/Landing-Page/)

> Acesse diretamente pelo link acima, sem precisar instalar nada!

| Tecnologia | Função no Projeto |
|------------|-------------------|
| ![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white) **HTML5** | Estrutura e organização do conteúdo da página |

## ✨ Funcionalidades

- Página de apresentação pessoal
- Layout moderno e responsivo
- Botão de contato direto via **WhatsApp**
- Design estilizado com HTML, CSS e JavaScript

